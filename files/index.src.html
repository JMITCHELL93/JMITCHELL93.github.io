EM Multipole Plotter


This simple PYTHON code allows electromagnetic multipoles to be graphed as vector plots. The code can of course be used for visulisation of electromagnetic multipoles and mode types but is also useful as an introduction to some basic PYTHON programming and techniques. The code has been edited from an existing code online for which the references are provided as comments in the script.

Download: PolePlotter.py
