    First Crabbing

    James Mitchell

    2018-06-03 20:52

    For two years now I have been working on Higher Order Modes (HOMs) in crab cavities as part of my PhD.

    I have been fortunate enough to be involved in the HL-LHC project (WP4) at a time when the first two manufactured cavities have been installed into the Super Proton Synchrotron (SPS) for preliminary tests with beam. This has meant that in addition to electromagnetic design work, I have been able to develop skills in cavity testing, RF measurements, high power RF and even some mechanical aspects associated with cavity manufacture.

    Read more about the first crabbing here.

    This Website

    James Mitchell

    2017-05-03 20:52

    As this is my first post, I thought it would be good to explain why I decided to make the site, provide some links to start your own and describe what I plan on using the site for in the future.

    This website was constructed in the persuit of impoving my PYTHON and HTML skills. The site was built using the static site generator Nikola which is written in python and was suggested to me by my good friend Matthew Allcock. Getting equiped with all you need to start your own static site is shown on Matthew's site and I very much promote referencing his site and using his blog posts to help guide you through the process.

    In addition to improving my python and HTML skills, the site has provided a briliant platform for sharing. Not only can I blog my thoughts and ideas, I can use the website as a repository for documentation, codes and other files which I want to be available for community use.

    I am sure that the purposes of the site will adapt and evolve over time, but this is one the advantages of having a site with blogging inbuilt. It can change with you and acts as a brilliant representation of both your personal character and work related achievements.
